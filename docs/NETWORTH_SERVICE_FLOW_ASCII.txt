│
├─ ProcessForeground.Bring(PoE2) → Delay(200)
│
├─ [stashOcrSearchRect.Width > 0]:
│   WindowsOcrTextLocator.TryFind(stashOcrText, searchRect)
│   найдено? → MoveTo(match) → ЛКМ
│   Delay(max(stashOpenDelayMs, 500))
│
└─ для каждой group в groups:
    │
    │  Items пустой? → пропустить
    │
    ├─ [NavTabRect задан] → MoveTo → ЛКМ → Delay(350)
    ├─ [TabRect задан]    → MoveTo → ЛКМ → Delay(500)

    │   PoeNinjaPriceService.GetPrice(itemName) → price.DivineValue
