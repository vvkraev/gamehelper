# GameHelper

Десктопное приложение для автоматизации крафта предметов в **Path of Exile 2**.

Выполняет циклы применения валютных сфер (Chaos Orb, Exaltation Orb, Augmentation/Annulment Orb) к предмету с автоматической проверкой нужных аффиксов по тексту из буфера обмена — без ручного повторения одних и тех же кликов до сотен раз.

---

## Требования

- **ОС:** Windows 10 версия 1809 (build 17763) или новее / Windows 11
- **Среда выполнения:** [.NET 10 Runtime для Windows](https://dotnet.microsoft.com/download/dotnet/10.0)
- **Для сборки:** .NET 10 SDK, Visual Studio 2022+ или Rider

---

## Быстрый старт

```
1. Скачать релизный архив → распаковать → запустить GameHelper.exe
2. Задать области экрана (кнопки «Задать область» для каждого слота)
3. Выбрать условие остановки через «Условие крафта»
4. Нажать «Старт»
```

Настройки сохраняются в `settings.json` рядом с `.exe` автоматически.

---

## Режимы крафта

| Режим | Что делает |
|---|---|
| **Chaos Craft** | Shift+ПКМ по Chaos Orb → ЛКМ по предмету → проверка аффикса. Повторяет до совпадения условия или лимита операций. |
| **Exaltation Craft** | Применение Exalted Orb с управлением омнами (Sinistral / Dextral / Greater). Поддерживает автоактивацию омена перед экзальтом. |
| **Aug + Annul** | Чередование Augmentation Orb и Annulment Orb для добавления/удаления модификаторов. |
| **Sharpen** | Заточка предметов по сетке инвентаря. |

## Настройка областей экрана

Все области задаются через полноэкранный выделитель — зажать ЛКМ и нарисовать прямоугольник поверх нужного элемента UI игры. Ручное редактирование координат не нужно.

**Основные области:**
- **Область орба** — куда кликать для применения сферы (Shift+ПКМ)
- **Область предмета** — куда кликать для применения орба (ЛКМ)
- **Область омена (Sinistral / Dextral / Greater)** — ячейки омена в инвентаре
- **Область инвентаря валют / ритуала** — для режима Aug+Annul

---

## Условие остановки крафта

Задаётся в окне «Условие крафта». Поддерживает:

- **Проверку подстроки** — крафт остановится, если в тексте предмета есть нужная строка
- **Числовые пороги** — шаблон с `n` на месте числа: например `increased Physical Damage n` с порогом `120` остановит крафт при значении ≥ 120
- **Логику И/ИЛИ** — несколько условий внутри варианта объединяются через И; несколько вариантов — через ИЛИ

Текст предмета считывается через `Ctrl+Alt+C` (PoE2 копирует в буфер обмена).

## Параметры

| Параметр | По умолчанию | Описание |
|---|---|---|
| `MouseActionDelayMs` | 80 мс | Задержка между действиями мыши |
| `ClipboardDelayMs` | 220 мс | Задержка после Ctrl+C перед чтением буфера |
| `MaxOps` | 20 | Максимальное число операций крафта |
| `StepConfirm` | false | Пошаговый режим с подтверждением каждого шага |
| `TraceInput` | false | Подробное логирование всех действий ввода |

---

## Логи

Все операции записываются в папку `Log/` рядом с `.exe`:
- `session_YYYY-MM-DD.log` — лог текущей сессии
- Отдельные файлы для каждого запуска крафта

## Важно

Приложение взаимодействует с игрой только через стандартные механизмы ОС: эмуляция мыши/клавиатуры и буфер обмена. Никакие файлы игры не читаются и не изменяются. Сетевые запросы не выполняются.
